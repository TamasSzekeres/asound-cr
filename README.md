# asound-cr

ALSA - libasound bindings for Crystal language.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  asound-cr:
    github: TamasSzekeres/asound-cr
```

## Usage

```crystal
require "asound"
```
